## üìã Quick Reference

### Installation by Category

```bash
# Install all from category
yourname-ui add --category hero          # All 175 hero components
yourname-ui add --category pricing       # All 150 pricing components
yourname-ui add --category testimonial   # All 120 testimonial components

# Install specific range
yourname-ui add hero-01..hero-10        # Hero 1-10
yourname-ui add pricing-01..pricing-20  # Pricing 1-20

# Install by tag
yourname-ui add --tag animation         # All animated components
yourname-ui add --tag ecommerce         # All e-commerce components
yourname-ui add --tag dark-mode         # All dark mode components
```

### Popular Combinations

```bash
# SaaS Landing Page
yourname-ui add hero-01 feature-03 pricing-07 testimonial-02 cta-01 footer-01

# E-commerce Store
```
